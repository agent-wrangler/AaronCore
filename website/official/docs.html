A map to the memory system

Start with the parts users can feel: flashback, time recall, persona continuity, and memory hygiene.

Core terms

Flashback = old-thread resonance. Recall = summarize conversation by time. Persona = preferences, relationship, and style continuity. Hygiene = filter noisy memory before it sticks.

See memory proof
Overview

What the memory system is

A layered memory stack that writes, filters, recalls, and reinjects memory back into conversation.

Flashback

Flashback

Old repair threads, emotional cues, or project states can resurface as natural hints instead of keyword dumps.

Recall

Time recall

Ask what you talked about today, yesterday, or last week and get a recap from chat history plus memory snippets.

Persona

Persona continuity

Preferences, dislikes, city, relationship posture, and interaction rules feed back into how AaronCore replies.

Hygiene

Memory hygiene

Think blocks, low-signal turns, and polluted context are filtered so long-term continuity stays cleaner.

Retrieval

Retrieval posture

Flashback is driven by resonance and continuity, not just plain keyword hits.

Boundaries

Boundaries

The memory layers stay split by role so facts, experiences, rules, and knowledge do not collapse into one bucket.
