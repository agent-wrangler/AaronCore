# 项目总览

最后更新：2026-03-13

NovaCore 是一个本地桌面 AI 工作台。当前已经收口到一条主链：

`desktop.py -> agent_final.py -> http://localhost:8090/ -> output.html`

## 当前有效入口

- `desktop.py`：桌面启动器
- `agent_final.py`：当前唯一后端主入口
- `output.html`：当前唯一前端页面

## 当前模块分工

- `brain/`：表达层、LLM 调用、本地人格兜底
- `core/router.py`：规则路由，当前优先于 LLM 路由
- `core/executor.py`：技能执行封装
- `core/skills/`：当前技能目录
- `memory/`：记忆访问模块
- `memory_db/`：当前统一持久化目录

## 当前后端接口

- `GET /`：返回前端页面
- `POST /chat`：聊天主接口
- `GET /stats`：统计信息
- `POST /stats`：更新统计
- `GET /nova_name`：当前名称
- `GET /skills`：技能列表
- `GET /history`：最近历史
- `GET /memory`：记忆时间线
- `GET /health`：健康检查

## 当前状态约定

- `memory_db/` 是当前统一状态目录
- `memory/` 只保留兼容读取，不再作为主写入目录
- `docs/99-归档-archive/` 中的文档属于历史记录，不代表当前实现

## 启动方式

桌面模式：

```bash
python desktop.py
```

仅启动后端：

```bash
python agent_final.py
```

当前运行依赖至少包括：`fastapi`、`uvicorn`、`requests`、`pydantic`、`pywebview`。
