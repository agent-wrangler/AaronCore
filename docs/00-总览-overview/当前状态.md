# NovaCore 当前状态

更新时间：2026-03-13

## 一句话结论

项目当前是一个“单体后端 + 单页前端 + 桌面壳”的本地 AI 工作台，已经统一到 `agent_final.py` 这条主链，不再以 `agent.py` 为当前版本。

## 当前主链

```text
desktop.py
  -> agent_final.py
  -> http://localhost:8090/
  -> output.html
```

## 当前有效文件

- `desktop.py`：桌面启动器
- `agent_final.py`：当前唯一运行入口
- `output.html`：当前唯一前端页面
- `brain/__init__.py`：表达层和 LLM 兜底
- `core/router.py`：规则路由
- `core/executor.py`：技能执行器
- `core/skills/`：当前技能目录
- `memory_db/`：统一状态目录

## 这次收口后的关键变化

- 路由改为“`core.router` 优先，LLM 路由兜底”
- `agent_final.py` 已补齐 `/skills`、`/history`、`/health`
- `msg_history.json` 与 `stats.json` 统一写入 `memory_db/`
- 技能执行成功后会回写 `evolution.json`
- `long_term.json` 兼容 `summary` / `content` 两种字段格式
- `stock` 技能已从占位回复改成真实行情查询，当前支持常见美股代码、常见中文别名、A股 6 位代码，以及 A股/美股指数概览
- `L8` 当前已接回主链的安全版：只做“后台搜索 -> 知识沉淀 -> 下次对话回流”，默认不开启自动生成技能代码
- `L7 -> L8` 现在已接通：当用户出现明显负反馈时，会先记录纠偏规则，再在后台触发“内部纠偏笔记 + 可选联网补学”的定向再学习

## 当前状态目录

- `memory_db/msg_history.json`：当前结构化消息历史
- `memory_db/conversation_history.txt`：文本对话历史
- `memory_db/stats.json`：统计数据
- `memory_db/persona.json`：人格/名称
- `memory_db/knowledge.json`：技能知识
- `memory_db/knowledge_base.json`：自动学习沉淀
- `memory_db/evolution.json`：技能执行增长轨迹

## 仍然存在但不应再当主版本理解的文件

- `archive/legacy-entrypoints/agent.py`
- `archive/legacy-entrypoints/agent_simple.py`
- `archive/legacy-entrypoints/agent_fixed.py`
- `archive/legacy-entrypoints/agent_patch.py`
- `archive/legacy-ui/output_optimized.html`
- `archive/backups/` 中的显式备份文件

这些文件当前只具有“历史参考”意义，不代表实际运行路径。

## 当前主要风险

- `agent_final.py` 仍然偏大，后续应继续拆成 `api`、`services`、`storage`
- `desktop.py` 仍然硬编码了本机 Python 路径
- 前端 `output.html` 仍是单文件页面，维护成本会继续升高
- 归档已经清出来了，但后续最好继续补一份 `requirements.txt` 或 `pyproject.toml`

## 2026-03-13 记忆页补充规则

- `L1` 记忆粒子只在记忆页上方概览卡片展示总数量，不再进入下方事件时间线
- 记忆时间线当前展示 `L3 / L4 / L5 / L6 / L7 / L8`
- `L4` 当前统一命名为“人格图谱”，展示方式使用自然语言摘要，不再直接暴露 `persona.json` 的原始配置字段
- `L3` 已清理掉一批旧版“技能执行日志”脏数据；这类历史项原先误写进 `long_term.json`，现在已从 `L3` 主数据源迁出并归档到 `memory_db/long_term_legacy_skill_logs.json`
- 成长等级不再按固定 `100 EXP` 线性升级，而是采用“每级所需经验递增”的累计模型
- 当前成长经验权重为：`L1=min(count,150)*1`、`L3*4`、`L4*6`、`L5*18`、`L6*10`、`L7*8`、`L8*24`
- 当前升级曲线为：`Lv.1 -> Lv.2` 需要 `100 EXP`，之后每升一级额外增加 `20 EXP`
- 聊天主链现在会把最近对话显式整理成上下文传给 `brain.think()`，不再只靠当前一句话生成回复
- 对“那什么时候有啊 / 为什么 / 然后呢 / 这个呢”这类承接追问，默认沿用上一轮话题继续回答，不再优先反问“你指什么”
- 股票查询当前支持的典型问法包括：`NVDA股价多少`、`苹果股票现在多少`、`600519`、`今天大盘怎么样`、`今天美股怎么样`
- `L8` 权限配置文件是 `memory_db/autolearn_config.json`，当前默认允许联网搜索和写入知识库，但 `allow_skill_generation=false`
- 设置页现在已接入 `autolearn_config` 的可视化开关，当前可直接控制：总开关、联网补学、知识落库、负反馈纠偏补学、自动生成技能，以及几项触发阈值
- 侧边栏现在新增了“文档”栏，直接读取当前有效文档，不需要再去仓库目录里翻 `README / docs/00 / docs/10 / docs/20 / docs/30`
