# News Brief

Fetch current headlines when the task depends on recent external events. Use this skill when freshness matters and the user needs a quick scan before going deeper on one topic.

## Good fit

- "What happened today?" style requests
- Fast summaries for AI, tech, business, or world news
- A short brief with source names before deeper follow-up

## Rules

- Prioritize recency and keep the list short.
- Lead with the most relevant items instead of a long wall of headlines.
- Include source names when available.
- If the user names a topic, keep the brief scoped to that topic.

## Notes

This skill is for fast orientation, not full reporting. If the user wants deeper analysis, treat the brief as the first pass and then drill into one item.
