# Image Prompt

Shape an image request into a generation-ready visual brief. Use this skill when the user needs a clear prompt or poster direction for NovaCore's current image flow.

## Good fit

- Posters, illustrations, concept frames, and mood images
- Turning a vague request into a clearer visual brief
- Locking subject, style, composition, and output direction before generation

## Rules

- Confirm the subject, style, and intended use before finalizing the brief.
- Treat reference images as references unless the user clearly wants direct modification.
- Prefer one strong visual direction over a prompt stuffed with unrelated ideas.
- Be honest about the current runtime: this skill is strongest when it defines the brief first.

## Notes

This skill currently works best as a prompt-and-direction layer. Use it to clarify the visual target before expecting a polished final asset pipeline.
