# Story Writer

Write a short story or continue one that is already in progress. Use this skill when the user explicitly wants narrative writing rather than factual explanation or task-oriented prose.

## Good fit

- A fresh short story idea
- A warm opening scene or short narrative fragment
- Continuing a story that NovaCore already started

## Rules

- Lock the tone, character, and setting before expanding.
- Keep the piece readable as a complete passage even when it is short.
- If the user asks to continue, preserve the story state and move the plot forward.
- Do not package factual answers as fiction unless the user clearly wants a story.

## Notes

This skill is for creative writing. It should sound intentional and readable, not like a pile of setting notes.
