# Article Draft

Turn source material into a publishable first draft and optionally save it. Use this skill when the user wants a complete article draft, a rewrite, or a cleaner structure built from raw notes or current headlines.

## Good fit

- Writing a readable first draft from news material
- Rewriting a rough article into a clearer structure
- Turning loose points into a single article with a defined angle

## Workflow

- Start from the source material that is already available.
- Pick one angle instead of mixing unrelated points into a single draft.
- Deliver a complete draft before polishing line by line.
- Save to desktop only when the user asks for it or the workflow already calls for saving.

## Notes

This skill writes drafts. It should not invent missing facts or replace verification when the source material is incomplete.
